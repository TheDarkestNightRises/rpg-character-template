# RPGCharacterTemplate

##🎮 Project Description:
A UE4 mage character template. This mage can perform spells and heal.

##🎯 Features:
- Character can run
- Character can cast fireballs to throw at enemy
- Character has mana bar, regen bar
- Heal ability
- Pickups for mana / health
